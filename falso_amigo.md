# :star: falso amigo"

### Do que se trata? 

Um falso amigo em português é uma palavra que pode ser igual a uma em espanhol, mas seu significado é totalmente diferente. Temos que ter cuidado com os falsos amigos! :ojos:

## :star: NOME / SOBRENOMBRE / APELIDO

| --------- | ------- |
| nome | nombre |
| sobrenome | apellido |
